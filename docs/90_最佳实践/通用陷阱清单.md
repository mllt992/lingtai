# 通用陷阱清单

> 这是软件开发中**反复出现的坑**。设计/编码/测试前都默念一遍："我这次会不会重蹈？"
> 项目踩到新坑要追加到对应小节，让清单长期有效。

---

## 一、数据类陷阱

### 1.1 日期与时区
- **开闭区间不一致**：前端传 `2026-01-01 ~ 2026-01-31`，后端 SQL `BETWEEN start AND end`，丢失 31 号 23:59:59
  - 修复：明确"包含起始日 23:59:59" 还是 "次日 00:00:00"，三端一致
- **时区混乱**：服务器 UTC，用户 GMT+8，前端显示又用本地时区
  - 修复：传输用 ISO-8601 带时区；展示在边界层一次性转换
- **闰年 2/29**：日期计算忘记考虑
- **跨月/跨年**：每月计费、年度报表的"上月/上年"计算
- **DST 夏令时**：跨 DST 切换的日期计算

### 1.2 小数与百分比
- **重复转换**：DB 存 0.05，后端 DTO 转成 5（%），前端又乘 100 显示成 500%
  - 修复：转换只发生一次（前端展示 或 后端 DTO），其他层用同一种表示
- **精度丢失**：用 float 处理金额、百分比
  - 修复：用 Decimal / BigDecimal / 整数（分）

### 1.3 金额
- **单位不一致**：有的接口分有的接口元
  - 修复：项目内统一单位，DTO 字段名带单位（`amountInCents` / `priceYuan`）
- **负数 / 零 / 极大值**：折扣 100%、退款超原价、库存大于 21 亿（int 上限）
- **千分位 + 等宽字体 + 右对齐**：UI 显示金额的三件套

### 1.4 字符编码
- **CSV/Excel 中文乱码**：UTF-8 没有 BOM，Excel 不识别
  - 修复：导出 UTF-8 BOM 或导出 .xlsx 而非 .csv
- **中文文件名 URL**：未 encode 导致 404 或破折号
- **emoji / 特殊字符**：DB 字段是 utf8mb3 而不是 utf8mb4，写入失败

### 1.5 排序稳定性
- **null 排在哪**：不同 DB 默认不同
- **大小写敏感**：中英文混排
- **翻页串行**：分页排序字段不唯一，第 2 页可能与第 1 页重复
  - 修复：排序字段后追加 ID 作为 tiebreaker

### 1.6 空值
- `null` / `""` / `0` / `[]` 语义不同但代码常常混用
  - 修复：每个字段在 API 文档里写清"空时含义"
- UI 显示 `undefined` / `null` 字面量给用户看
  - 修复：边界处理在展示层

### 1.7 超长 / 超大
- 备注字段没限长，DB 报错；UI 没截断，破版
- 列表导出 10w 行卡死
- 上传文件无大小限制

## 二、接口类陷阱

### 2.1 分页
- **page 起始**：0 还是 1？前后端不一致直接错位
- **total 是否含被筛掉的**：分页接口的 total 经常不准

### 2.2 软删除
- 新写的查询忘记带 `is_deleted = false`
- 级联查询时关联表未同步过滤
  - 修复：ORM 全局过滤器 / 共享 SQL 片段 / 视图

### 2.3 列表→详情字段不一致
- 列表接口返回 `name = "张三"`，详情接口返回 `userName = "张三"`，前端缓存崩
  - 修复：DTO 复用，命名一致

### 2.4 批量接口
- 部分成功如何返回？事务边界没说明？
  - 修复：明确"全部成功 / 全部失败 / 允许部分成功"，按需返回 `successCount`、`failures`

### 2.5 长字段
- 备注 / 富文本超过 4000 字符，DB 列类型不够，写入失败
- 列表展示长字段不截断，破版

### 2.6 版本与兼容
- 加字段未通知调用方，老代码报"字段不存在"
- 删字段未走 deprecation，老代码直接报 500

### 2.7 幂等
- 创建接口没幂等，用户连点两下，订单出两份
  - 修复：`Idempotency-Key` 或业务唯一键

### 2.8 错误码不一致
- 同一类错误返回不同 code
- 前端拿到 code 不知道是什么（错误码字典缺失）

## 三、前端类陷阱

### 3.1 状态管理
- 父子组件双向绑定失效（框架特有 API 用错）
- 异步 setup 中 await 之后失去响应性 / inject 上下文
- store 与组件 state 双轨同步不一致

### 3.2 列表 / 表格
- rowKey 重复导致编辑串行
- 列宽未限制 / 长文本未截断 → 破版
- 排序状态切页面后丢失
- 大数据量未开虚拟滚动

### 3.3 表单
- 提交后没禁用按钮，连点产生重复
- 后端字段级错误未映射回字段
- 表单状态未在路由切换前确认（数据丢）
- 提交前未做客户端校验（用户体验差）

### 3.4 路由
- 守卫顺序问题：菜单未加载完跳转 → 404 / 闪烁
- 未登录访问受保护页直接白屏（应跳登录）
- 参数变化但页面不刷新（同路由不同参数）

### 3.5 网络
- 同一数据多处独立请求 → 多次拉
  - 修复：用 SWR / Query 库
- 切页面没取消上一个未完成请求
- 错误处理不统一（多处弹 toast）

### 3.6 上传
- 进度条不准 / 不可取消
- 大文件不分片，断网必从头
- 上传后未更新页面（用户不知道成功了没）

### 3.7 浏览器兼容
- 用了新 API 老浏览器不支持，没 polyfill
- Console 红色错误未处理

## 四、权限与安全类陷阱

### 4.1 功能权限只在前端做
- 前端隐藏按钮但接口可被构造请求调用
  - 修复：后端必须独立校验

### 4.2 数据权限缺失
- 列表过滤了，详情接口没过滤 → 构造 ID 越权读
- 列表过滤了，导出接口没过滤 → 拖库
- 列表过滤了，更新/删除接口没过滤 → 横向越权操作

### 4.3 字段脱敏在前端做
- 接口返回完整身份证，前端用 `***` 显示 → 一查 DevTools 全暴露
  - 修复：脱敏在 DTO 层

### 4.4 日志泄露
- 打印明文密码、token、身份证
- 错误堆栈含敏感参数被透传到前端

### 4.5 SQL 注入
- 字符串拼接 SQL（特别是动态字段名 / 表名场景）
  - 修复：参数化 + 白名单

### 4.6 输入校验
- 只有前端校验，后端裸接
- 后端校验只在 Controller 做，Service 未二次校验（绕过的危险）

### 4.7 密钥泄漏
- `.env` 误提交 git
- 配置写死在代码里
- 日志 / 错误信息打出来

## 五、并发 / 事务类陷阱

### 5.1 事务失效
- `@Transactional` 加在 private / final / static 方法
- 自调用（`this.xxx()`）不走代理
- 抛出非 RuntimeException 但未声明 `rollbackFor`

### 5.2 事务长事务
- 事务里调外部 HTTP / RPC → 连接长期占用，可能死锁
- 事务里发 MQ → MQ 发了但 DB 回滚（或反之）
  - 修复：Outbox 模式

### 5.3 并发写
- 没乐观锁 / 唯一索引，并发提交都成功
- 库存超卖
- 计数器更新丢失（`SET count = count + 1` 并发安全，`SET count = ?` 不安全）

### 5.4 缓存
- 双写不一致：DB 更新后忘删缓存 / 缓存先更新但 DB 失败
- 缓存击穿：热点 key 失效瞬间大量请求打 DB
- 缓存雪崩：大量 key 同时失效
- 缓存穿透：查询不存在的 key，每次都打 DB

### 5.5 异步
- 异步任务异常未捕获，进程挂
- 异步任务上下文丢失（用户、租户、traceId）
- 异步任务超时未限制

## 六、数据库类陷阱

### 6.1 索引
- 查询字段无索引 → 慢查询
- WHERE 字段做函数 → 索引失效（`WHERE DATE(created_at) = ?`）
- LIKE 前置通配 → 全表扫
- 联合索引顺序错 → 不命中

### 6.2 N+1
- ORM 关联懒加载在循环里触发
- 列表查询遍历每行又查关联表

### 6.3 迁移
- 大表加 NOT NULL 字段直接挂
- 改字段类型锁表
- 删字段没走"停写→观察→删"流程

### 6.4 备份与恢复
- 备份只备数据不备结构，恢复时挂
- 备份在同一台机器（机器挂同归于尽）
- 从未演练过恢复

## 七、性能类陷阱

### 7.1 接口
- N+1 查询
- 一次 fetch 全表
- 没分页
- 同步操作做了应该异步的事（导出大文件、发邮件）

### 7.2 前端
- 没用懒加载（路由 / 图片 / 组件）
- 大列表无虚拟滚动
- 频繁触发的事件没节流防抖
- 第三方库全量引入

### 7.3 构建
- 包体积超大
- 没开 tree-shaking
- 没拆 chunk

---

## 八、可观测性类陷阱

### 8.1 日志
- 关键路径无日志 → 出错无法定位
- 日志无 traceId → 跨服务无法串联
- ERROR 日志没堆栈 → 看不到根因
- 循环里 INFO → 撑爆磁盘

### 8.2 监控
- 接口耗时未埋点
- 错误率未告警
- 关键业务指标未仪表盘化

### 8.3 追踪
- 异步 / MQ / 定时任务丢失 traceId
- 跨服务 Header 未透传

---

## 九、规范类陷阱

### 9.1 命名
- 含义模糊（`data`、`info`、`flag`）
- 拼音命名
- 否定式（`isNotReady`）

### 9.2 注释
- 注释复述代码
- 注释过期未更新（比代码还误导）
- TODO 永远没人做

### 9.3 配置
- 魔法值散落各处
- 不同环境写死配置
- 配置变更不可回滚
